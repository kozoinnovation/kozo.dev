# SaaS収益計算機

SaaSビジネスのMRR/ARR/ARPUをリアルタイム計算。
ユーザー数とプラン配分から12ヶ月成長シミュレーションを可視化するWebツール。

## 🌐 URL

https://kozo.dev/tools/saas-calculator

## 🛠️ 技術スタック

| カテゴリ | 技術 |
|---------|------|
| フレームワーク | Next.js |
| グラフ | Recharts |
| スタイリング | Tailwind CSS |

## ✨ 機能

### 基本計算
- **MRR（月次経常収益）** - プラン料金 × ユーザー数
- **ARR（年次経常収益）** - MRR × 12
- **ARPU（ユーザー平均単価）** - MRR ÷ 総ユーザー数
- **ARPPU（有料ユーザー平均単価）** - 有料MRR ÷ 有料ユーザー数

### シミュレーション設定
- 総ユーザー数
- プラン料金（Free / Standard / Pro）
- ユーザー配分（スライダーで調整）
- 月次成長率
- 月次解約率
- 初月無料オプション
- Stripe手数料（3.6%）
- カスタムコスト（固定費・変動費）

### グラフ表示
- MRR推移（12ヶ月）
- ユーザー数推移
- プラン別ユーザー数（積み上げ棒グラフ）
- 累積純売上

### 出力機能
- **CSV** - 詳細データをスプレッドシートで分析
- **Markdown** - レポート用フォーマット
- **YAML** - 設定ファイルとして保存
- **シェア** - X, LINE, Threads, クリップボード

### その他
- 日本語 / 英語 切り替え
- プリセット（安定 / 成長期 / 急成長 / 停滞）
- 利益率の視覚化
