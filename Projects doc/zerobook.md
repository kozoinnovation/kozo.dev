# ZeroBook

予約システムSaaS。Webサイトに1行のコードを埋め込むだけで、予約フォームを設置できます。

## 🌐 URL

https://zerobook.app

## 📱 対応プラットフォーム

- Web
- iOS
- Android

## 🛠️ 技術スタック

| カテゴリ | 技術 |
|---------|------|
| フロントエンド | Next.js |
| バックエンド | Supabase |
| iOS | SwiftUI |
| Android | Jetpack Compose |

## ✨ 特徴

- **1行埋め込み（Shadow DOM）** - 親サイトのCSSに影響されない独立したフォーム
- **ダブルブッキング完全防止** - リアルタイムで予約状況を同期
- **リアルタイム同期** - 複数端末間でのデータ同期
- **AIメニュー取り込み** - 画像からメニューを自動認識

## 🎯 ターゲット

- 美容室・サロン
- クリニック・医院
- 飲食店
- その他予約が必要なサービス業
